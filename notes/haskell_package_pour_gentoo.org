#+title: Haskell package pour Gentoo
#+filetags: personal haskell gentoo

Génerer l'ebuild avec hackport
#+begin_src sh
hackport update
hackport merge diagrams-graphviz
#+end_src
Cela va générer le paquet dans dev-haskell (repo haskell)
Tester l'installation
#+begin_src
emerge -1 diagrams-graphviz
#+end_src
